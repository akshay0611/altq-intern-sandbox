# AltQ Design System (Light Version)

## 1. Color Palette
Reflecting a premium, modern salon aesthetic.

### Primary Brand
*   **Primary Black:** `#0F172A` (Slate 900) - Used for Text & Primary Buttons.
*   **Primary Gold:** `#D4AF37` (Metallic Gold) - Used for Accents, Stars, & Highlights.
*   **Soft Gold:** `#F3E5AB` (Champagne) - Used for backgrounds on active states.

### Functional Colors
*   **Success Green:** `#10B981` (Emerald 500) - Positive sentiment.
*   **Error Red:** `#EF4444` (Red 500) - Critical errors/Negative sentiment.
*   **Neutral Gray:** `#F1F5F9` (Slate 100) - Backgrounds.
*   **Border Gray:** `#E2E8F0` (Slate 200) - Borders.

## 2. Typography
Please use **Inter** or **Outfit** (Google Fonts).

*   **Headings:** semi-bold / bold
*   **Body:** regular
*   **Monospace:** Fira Code (for IDs)

## 3. Shadows (Tailwind mapping)
*   **Card Shadow:** `shadow-lg` (soft diffusion)
*   **Hover State:** `shadow-xl` + `scale-105` (micro-interaction)
