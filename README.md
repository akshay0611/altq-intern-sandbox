# AltQ Reputation Module (Internship)

Welcome to your internship sandbox! This repository contains the scaffold for the Reputation Module project.

## Structure

*   **/frontend**: React application (Vite + TypeScript)
*   **/backend**: Express API (Node + TypeScript)

## Getting Started

### 1. Setup Frontend
```bash
cd frontend
npm install
npm run dev
```

### 2. Setup Backend
```bash
cd backend
npm install
npm run dev
```
